Las página web pueden ser estáticas o dinámicas

  • Página estáticas:
    • Fueron las primeras en desarrollarse
    • Aún se utilizan para la presentación de datos
    • Son fáciles de crear, instalar y mantener
  • Página dinámicas:
    • Permiten mayor interacción con el usuario
    • Son útiles para procesar datos ingresados en formularios y otras funcionalidades avanzadas
