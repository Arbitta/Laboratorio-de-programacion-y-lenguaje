  1. Qué es el desarrollo web?

    2. El desarrollo web es el proceso de crear y mantener sitios y aplicaciones en internet. Implica el uso de tecnologías como HTML, CSS y JavaScript, además de frameworks y herramientas avanzadas que facilitan la construcción de plataformas más eficientes y atractivas.

  2. Tipos de páginas web

    • Páginas estáticas
    • Páginas dinámicas

  3. Elementos esenciales de una página web

    1. Estructura(HTML)
    2. Estilo (CSS)
    3. Interactividad (JavaScript)

  4. Importancia del diseño web

    • ✅ Ser intuitivo y fácil de navegar
    • ✅ Adaptarse a distintos dispositivos (diseño reponsivo)
    • ✅ Cargar rápidamente y ofrecer una buenas experiencia de usuario
